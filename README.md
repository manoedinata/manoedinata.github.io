# [manoedinata.github.io](https://manoedinata.com)

![image](https://github.com/user-attachments/assets/2d59048d-d8a2-47bf-8620-4cdaa8fb2325)

My personal homepage, serving as my portfolio, projects showcase, and blog. Built with Bootstrap 5 and Astro, deployed on Netlify (despite the repository name which is meant for GitHub Pages). Additionally, Decap CMS is configured to be used as the content manager, primarily for projects & blog posts.

The website design is based on [Personal](https://bootstrapmade.com/personal-free-resume-bootstrap-template/) by [BootstrapMade](https://bootstrapmade.com), adjusted accordingly for use with Astro.

## Running

You know this already, don't you?

```bash
npm install
npm run dev
```
